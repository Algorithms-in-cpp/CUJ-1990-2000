Keywords: supplement, Java, Swing, multithreading, RMI, Collections  

The code listings for the Java Solutions supplement to the December 1999
C/C++ Users Journal include source code for Chuck Allison's article,
"The Java 2 Collections," Peter Meehan's article, "Handling Non-User
Events in Swing," and Petru Marginean's article, "Communication Between
Browswers Using RMI."
