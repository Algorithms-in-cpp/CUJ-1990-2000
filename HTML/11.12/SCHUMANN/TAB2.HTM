Table 2 Functions for returning arguments from C functions

_ret (void);                 Returns nothing (like C's void)
_retc (char far *);          Returns string
_retclen (char far *, int);  Returns string with explicit length
_retni (int);                Returns int as a Clipper numeric
_retnl (long);               Returns long as a Clipper numeric
_retnd (double);             Returns double as a Clipper numeric
_retl (int)'                 Returns int as a Clipper logical
_retds (char far *);         Returns YYYYMMDD string as a Clipper date
