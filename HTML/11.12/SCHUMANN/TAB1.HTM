Table 1 Functions for passing arguments to C functions

char far *_parc (int)   Pointer to a string argument
int _parclen (int)      Length of string argument
int _parni (int);       Numeric argument as a (16-bit) integer
long _parnl (int);      Numeric argument as a long (32-bit) integer
double _parnd (int);    Numeric argument as a double (32-bit)
int -parl (int);        Logical argument as a 1 or 0 integer
char far *_pards (int)  Pointer to a date argument read as a string of
                        YYYYMMDD
