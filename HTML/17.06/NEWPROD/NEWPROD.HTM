Technology Builders Integrates Caliber-RM with Bender & Associates SoftTest 5.3

Technology Builders, Inc. has announced the integration of Caliber-RM with Bender & Associates SoftTest 5.3. Caliber-RM is an enterprise software tool that enables project teams to define, manage, and communicate changes to application and system requirements. The integration enables project teams to link requirements in Caliber-RM to the functional test cases generated by SoftTest 5.3, and to determine the functional coverage during the testing process for an application, regardless of platform or language.

SoftTest 5.3 analyzes the requirements for identifying the functional variations to test and then packages them into a minimum set of logical tests providing maximum functional coverage. It also identifies logical inconsistencies in the specifications.

Research Systems Introduces RiverTools

Research Systems has released RiverTools, a commercial data-modeling application for terrain and river networks. RiverTools is designed to help hydrologists view and process digital elevation model data. It contains a full-featured topographic and hydrologic analysis application and combines comprehensive terrain and river network analysis with a point-and-click user interface.

RiverTools calculates the location and magnitude of streams and saves this information into a hierarchical river tree file for display and additional analysis. While creating the stream model, RiverTools automatically determines over a dozen characteristics for every node in the river tree. These characteristics include upstream area, channel lengths, drops and slopes, and a variety of topologic data.

RiverTools is available for Windows 95/98/NT, Power Macintosh, Linux, and other versions of Unix. Prices start at $999.
