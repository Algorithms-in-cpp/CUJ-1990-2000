Objective Interface Unveils ORBexpress Real-Time for C++

Objective Interface has released ORBexpress Real-Time for C++, a product that provides architectural enhancements for building distributed real-time and embedded systems. In addition to full IIOP support, ORBexpress also incorporates a Micro-ORB architecture that lets developers replace key components of the ORB itself.

The ORBexpress Resource Allocation Control Environment (RACE) also allows the developer to manage the impact of limited resources on service quality. The small memory footprint of ORBexpress Real-Time for C++ is useful for memory-constrained devices, like routers and switches. Another capability, Distributed Priority Inheritance, reduces the intersystem priority inversion problems often found in distributed real-time systems.

Crossware Offers ANSI C Compiler with Support for Latest ColdFire Chips

Crossware has released an ANSI C compiler for Motorola's ColdFire range of 32-bit microcontrollers. All ColdFire series 2 and series 3 chips are supported, including the hardware divide and remainder instructions and the high-speed Multiply and Accumulate (MAC) unit. The MAC unit, which can perform a 16-bit multiply in a single cycle, has been enhanced to perform fixed-point fractional arithmetic.

The compiler generates full debug records allowing debugging tools access to the program source-level code and data structures. Legacy software written for the Motorola 68000 family of microprocessors is compatible with the ColdFire architecture. At the assembler level, the Crossware relocatable assembler is compatible with the porting tool from MicroAPL, which converts 68000 assembler code into ColdFire compatible assembler code. At the C level, the ANSI standard ensures that compatibility issues are minimized.
