SYWARE Announces Dr. DeeBee ODBC Tools v3.0

SYWARE, Inc. has announced Dr. DeeBee ODBC Tools v3.0, a suite of utilities used to monitor the operation of ODBC-enabled applications and drivers. With Version 3.0 the tools Info, Peek, Timer, and Spy now support both ODBC 2.x and 3.0.

Dr. DeeBee Tools is a suite of seven utilities: Check semantically checks ODBC applications at runtime; Peek records ODBC calls made by an application; Timer counts the number and length of ODBC calls an application makes; Test runs regression tests on ODBC drivers; Spy is a redistributable version of Peek for use by product support organizations; Replay replays logs generated by Spy; and Info displays description and characteristics of a driver.

Dr. DeeBee Tools are packaged in two bundles: the Starter Bundle (Info, Peek, and Timer) costs $199 for 16-bit and $299 for 32-bit; the Super Bundle (all seven tools) costs $399 for 16-bit and $499 for 32-bit.
