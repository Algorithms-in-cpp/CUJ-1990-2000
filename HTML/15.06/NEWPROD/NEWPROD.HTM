StratosWare Releases MemCheck 5.0 for Windows

StratosWare Corporation has announced MemCheck 5.0 for Windows, a runtime error detection tool for applications running under Windows 95 and Windows NT. Software Research Announces TCAT for Java

Software Research, Inc. has announced TCAT for Java, a coverage analyzer that reviews a coverage trace and presents the results in graphic, textual, and tabular reports.

TCAT for Java provides both branch and method-invocation (call-pair) coverage in a single test run to identify which part of a Java applet's code has been tested, both at the unit/object level and the system integration level. Branch coverage is used for unit and system testing, tracking the number of times each segment of a method has been exercised. Call-pair coverage tests integration by determining if each method invocation has been exercised. Testers can then efficiently concentrate on untested code, reducing the likelihood of defects and improving product quality.

NewMonics Announces PERC

NewMonics Inc. has announced PERC virtual machine for sublicensing. By replacing Sun Microsystem's Java virtual machine with the NewMonics product, developers can take advantage of NewMonics' real-time garbage collection technologies and ROM-targeting, pre-runtime code optimizer.

PERC's garbage collector incrementally defragments memory by copying in-use objects to contiguous memory locations to coalesce multiple small free segments into larger free segments. It does this under strict real-time execution constraints, enabling high-priority application tasks to preempt the garbage collector and offering throughout guarantees for the rate at which each real-time activity's memory can be reclaimed and reallocated.
