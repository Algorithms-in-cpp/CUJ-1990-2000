Figure 1: User commands recognized by dumsocks client

TERM passwd               - terminate dumsocks server
STATUS                    - get status of dumsocks server
CLIENT.LOG.OFF            - turn client log off
CLIENT.LOG.ON [fname]     - turn client log on
CLIENT.LOG.STATUS         - report client log status
SERVER.LOG.OFF            - turn dumsocks server log off
SERVER.LOG.ON [fname]     - turn dumsocks server log on
SERVER.LOG.STATUS         - report dumsocks server log status
CONNECT connect_string    - connect to another dumsocks server
SOCLOC.GET.LIST           - report all socket servers
SOCLOC.CONFIG.GET.LIST    - report all socloc server
SOCLOC.GET                - report on current socloc server
SOCLOC.LOG.OFF            - turn socloc server log off
SOCLOC.LOG.ON [fname]     - turn socloc server log on
SOCLOC.LOG.STATUS         - report socloc server log status
SOCLOC.VERSION            - report socloc server version
VERSION                   - report dumsocks server version
OFF                       - terminate dumsockc client
