MAILERS Software Introduces Address Object

MAILERS Software has introduced Address Object, a COM object that is part of a group of products collectively called the Data Verification and Enhancement Tools. Address Object enables programmers to verify addresses for website orders, reduce keystrokes during data entry, and use clean data for data mining and database marketing.

Address Object verifies addresses on-the-fly or in batch. It compares the addresses on the user's list to the USPS National Address Database. Once an address is corrected or verified, a ZIP+4 code is added. Additional data linked to the address can be appended to the record, including latitude and longitude, time zone, county and MSA data, and congressional district.

Address Object works with programs written in a variety of languages, including Active Server Pages, C++, Delphi, JScript, Progress, SQL stored procedures, Visual Basic, and VBScript. A free evaluation copy is available from the company's website.

Softwrap Unveils Softwrap "Wrapper"

Softwrap has released Softwrap, a wrapper for authors and distributors that prevents piracy and manages the digital distribution process. Softwrap prevents piracy by encrypting any 32-bit Windows compatible .exe application. Once encrypted, the file can be executed only for an author-stipulated trial period before the user will be prompted to purchase. If a customer wants to use the application beyond the trial period, they must purchase it or discontinue its use. Additional security features include checking for system clock rollbacks, built-in hack detection, and the prevention of wrapping already wrapped files.

Softwrap manages the digital distribution process by offering a service that manages the e-commerce activities for authors/distributors. Softwrap employs a monthly, user credit-card verification process that keeps fraud to a minimum. Payment for the application is billed to the customer's credit card. Softwrap deducts service fees from the revenue and sends the balance to the author in the form of a monthly check. Revenue after fees depends on the retail cost of the application and number sold.

Softwrap may be downloaded for free from the company's website.

For more information, contact Softwrap, 800.221.8984 or +44.207.845.6504; fax +44.207.240.2535; www.softwrap.com.
