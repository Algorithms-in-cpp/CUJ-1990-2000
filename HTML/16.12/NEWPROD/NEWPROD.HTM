Pacific Firmware Marketing Unveils BurnLock

Pacific Firmware Marketing has released BurnLock, an Internet solution for piracy control. BurnLock makes cracking and hacking a trial version of an application difficult because BurnLock data is integrated directly in predefined positions in the EXE file. To succeed in breaking BurnLock, a user must decompile the object code; discover the positions of the encrypted, randomly placed BurnLock data in the source code; erase and replace the BurnLock data; and recompile the code again.

BurnLock is a complete developer toolkit for constructing trialware and shareware versions of any application. BurnLock costs $499 per product per year.
