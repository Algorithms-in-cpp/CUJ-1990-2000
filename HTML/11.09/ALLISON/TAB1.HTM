Table 1 Array-to-Pointer conversions for a 2-dimensional array

Expression  Type                         Value
-----------------------------------------------
a           array of 3 arrays of 4 ints  0x0
a + 1       pointer to array of 4 ints   0x8
a[1]        array of 4 ints              0x8
a[1] + 1    pointer to int               0xA
a[1][1]     int                          5
(Hexadecimal values are addresses relative to a)
