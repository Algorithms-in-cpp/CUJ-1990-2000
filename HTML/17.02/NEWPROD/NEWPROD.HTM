Cygnus Solutions Unveils Introspect

Cygnus Solutions has released Introspect, a software-based debugger for real-time embedded applications. Introspect records the state of program execution and events for analysis at a later time without hardware, such as logic analyzers or emulators. Introspect provides non-disruptive data collection for real-time applications.

Unlike trace data collected on a host computer, Introspect provides target-side data collection, gathered without slowing down the execution of the target program. In addition, Introspect provides the capability to arrange trace experiments and then review and analyze results later. It allows developers to step through an application and see what may have taken hours or weeks to occur.

Datalight Announces FlashFX for Windows CE

Datalight Inc., has released FlashFX for Windows CE to BSQUARE Corporation as a part of its total solution for Windows CE. To accomplish hard-disk emulation, FlashFX incorporates Variable Block Flash, a patent-pending data format. In a manner similar to a conventional hard disk, the VBF format emulates small sectors, although the flash memory may contain sectors as large as 128K.

To maximize flash-memory lifetime, FlashFX includes wear-leveling algorithms. FlashFX provides true wear leveling that ensures the entire flash memory is used evenly for both static and dynamic data areas.
