Pure Software Ships Purify for Windows NT

Pure Software has released Purify for Windows NT. Purify for Windows NT provides a way to automatically pinpoint run-time errors and memory leaks throughout C and C++ applications, including the ActiveX controls, OLE components, third-party DLLs, and libraries that are the building blocks of 32-bit Windows applications.

Based on Object Code Insertion (OCT), a patented technology from Pure Software, Purify can locate hard-to-find errors and memory leaks. Because OCT does not require access to source code, Purify can locate errors throughout the third-party components used to build many Windows applications. Purify can check multiple applications simultaneously, and supports multi-threaded and multiprocess applications, as well as applications running on multiprocessor machines.

Prices for Purify for Windows NT start at $498. A subscription including upgrades and maintenance is $748. European, Japanese, and Asia Pacific pricing is available by contacting Pure Software directly.

Subtle Software Releases CGEN 3.0

Subtle Software, Inc. has released Developer Edition v3.0 of Subtleware C++ Class Generator (CGEN), a Windows tool that creates persistent C++ classes from relational databases. CGEN eliminates the need to hand-write mapping code that transfers object data between application classes and databases. With a click of a button, it generates and implements C++ methods that read and write application objects. The resulting class definitions and implementations contain complete, error-free storage and retrieval methods.

Once a persistent C++ class has been generated using CGEN, users can refine the class and build applications in a manner similar to their normal development process. The resulting C++ application is then able to read and write data using the original database schema. CGEN works with and extends the capabilities of database class libraries by automating the implementation of read, write, and remove methods for user-defined types. Developers can specify the use of DBTools.h++, MFC, Borland's ClassLib, or the included Subtleware SQLExec database class library.

CGEN runs under Windows 3.x, NT, and 95. Introductory price is $399.

On Time Announces RTTarget32

On Time Marketing has announced RTTarget-32, a cross-platform development system for 32-bit embedded systems on the Intel 386 platform. RTTarget-32 allows 32-bit programs developed for Windows 95 or NT to run without an operating system on an Intel 386, 486, or Pentium Processor. Popular C/C++ compilers from Borland, Microsoft, and Watcom are supported, and DOS, Windows, Windows 95, or Windows NT systems can be used as a host.

RTTarget-32 provides compact boot code (6Kb) to start the target system from floppy disk, hard disk, EPROM disk, flash disk, or directly from ROM. Subsequently, the application program either runs directly from ROM or is loaded from disk. The application can also be downloaded using a serial link during the development phase. The compact boot code enables execution of full 32-bit applications in as little as 16K of memory.

RTTarget-32's Locator processes the PE-files created by all Win32 linkers. A configuration file enables versatile mapping of the application program to the target hardware. Code and data may be separated into ROM and RAM regions, or copied from ROM to RAM at initialization time. The 386's memory protection capabilities are fully supported.

RTTarget-32 provides a comprehensive runtime library on the target, and emulates part of the Win32 API, thereby supporting software originally developed for DOS or Win32. It also supports the C/C++ runtime system for memory management and I/O, and provides additional functions for real-time processing.

RTTarget-32 supports cross debugging with Borland's 32-bit debugger TD32, and provides some debugging functions normally seen only in in-circuit emulators.

An RTTarget-32 developer's license is priced at $1,700; complete source code is available for $1,000. There are no runtime royalties.
