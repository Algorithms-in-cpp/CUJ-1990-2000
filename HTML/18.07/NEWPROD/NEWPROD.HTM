Quma Software Announces QVCS v3.4

Quma Software has released QVCS v3.4, a Windows 95/98/NT4/2000 version control system. QVCS v3.4 automates the tracking of files as they change during the development of an application. By requiring developers to log in, QVCS v3.4 prevents more than one copy of a file from being edited at one time. After log in, QVCS v3.4 displays a warning if a writeable copy of the file already exists and denies access to the file if another developer is already working on the file.

QVCS v3.4 provides a number of reports, such as revisions created after a release and revisions created between two dates. Also, QVCS v3.4 can be integrated into any IDE that supports Microsoft's Source Control API.

North Coast Idea Company Releases Vector Image Interface 2.0

North Coast Idea Company, Inc. has released VIX (Vector Image Interface) 2.0, a C++ Windows developer tool for merging vector images with dynamic content. When the VIX engine renders an image, it calls back to the application or component each time it encounters a rectangle. The implementation is called with a rectangle id, a device context, rectangle dimensions, etc. This enables the user to fill in the dynamic content at run time.

Vector Image Interface 2.0 includes VIX Workshop, a visual tool for creating and working with vector images. VIX Workshop extracts the vector image from a drawing tool (such as Freehand, Corel Draw, or PowerPoint) and creates a header file that associates the rectangles in the image with #define tokens.

Vector Image Interface 2.0 costs $195 and may be purchased online at the company's website.

For more information, contact North Coast Idea Company, Inc., sales@ncidea.com; www.ncidea.com.
