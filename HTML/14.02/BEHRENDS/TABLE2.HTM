Table 2: ttymon usage syntax

Usage: ttymon[-?][-s][-r][-l file][-c][-d][-t#][-o#][-b#] port1 port2

-? Print option summary
-s Disable display of session on stdout (default=display enabled)
-l Specify file for log output (Has no effect on display to stdout)
-o Write an output entry at regular intervals of independent data
-d Put process in daemon state
-r Disable raw mode on ports

Buffer size and timeout options:

-t Buffer size threshold for forcing a write before timeout
-o timeout value in milliseconds
-b Input buffer sizes
