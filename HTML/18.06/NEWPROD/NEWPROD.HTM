Etnus Ships TotalView 4.0 Parallel Debugger

Etnus has released TotalView 4.0, a parallel debugger that supports multiple development platforms for both traditional Unix and Linux. New in version 4.0 is support for additional compilers, including SunPro 5.0 and Guide OpenMP cross-platform compilers on the IBM, SGI, Compaq, and Intel Linux platforms. TotalView 4.0 also supports 64-bit IBM AIX applications with the ability to debug both 32- and 64-bit programs in a single session.

The GUI-based, single- and multi-process debugger enables developers to unravel and control multiple threads and processes running on single or multiple processor systems. The debugger automatically acquires related information on processes and threads and graphically displays data arrays. Users can dive into program objects and structures for more detail.

Etnus offers licenses for two or more processors and is introducing a one-user, one-processor licensing option. TotalView 4.0 licenses start at $750.

Green Hills Introduces ISIM

Green Hills Software has introduced ISIM, an advanced RTOS simulation environment for the INTEGRITY real-time operating system. ISIM is a PowerPC instruction set simulator that provides support for PowerPC systems running the INTEGRITY RTOS, including virtual memory facilities, exception handling, and debugging across a LAN. ISIM enables designers to develop and test INTEGRITY-based embedded applications on PCs or workstations without requiring access to actual target hardware.

ISIM is tightly integrated with the MULTI debugger, enabling developers to debug a simulated application on the host in the same way they would on the target. ISIM supports both freeze-mode and run-mode debugging.

ISIM is available for use with INTEGRITY for $2,500 per user.
