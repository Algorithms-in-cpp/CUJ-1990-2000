Galapagos Releases Adaptive Object Storage v1.0

Galapagos Software, Inc. has begun shipping the first product based on Associative Database Technology (ADB), called Adaptive Object Storage v1.0 for C++. ADB, a new technology for finding, organizing, and storing data, mimics the associative capabilities of the human mind. Galapagos Software developed ADB to address today's information overload problem caused by the simultaneous convergence of multiple technologies.

ADB Technology is based on three core capabilities. First, it supports collections of values and bi-directional linking between these collections, allowing additions and deletions to these collections at any time. Second, it offers "category-item" and "recency" models that allow information to be organized based on inclusion in a category and on a specified time period. Third, it supports a unique "dynamic hierarchy" navigational model, which allows users to build a hierarchy on demand. Points of the hierarchy are not fixed, but are decided on by the user while performing a search or organizing information.

Adaptive Object Storage v1.0 costs $300.

TracePoint Announces Visual Coverage

TracePoint Technology, Inc. has announced Visual Coverage, a Windows-based testing tool that provides code coverage analysis throughout a product's development-test-delivery cycle. With Visual Coverage, developers can view coverage on any portion of their code and generate a report on the percent executed, as well as the specific areas that need further testing.

For each test run, developers can select various types of coverage. Function coverage shows which functions have and have not been executed. Class coverage allows the developer to view coverage by classes and by methods. Code coverage identifies which instructions have and have not been executed. Line coverage maps completed execution to specific lines of source code. Branch coverage shows which logical branches were and were not taken. Multiple condition coverage shows whether each condition was covered with both the true and false values. Call Pair coverage identifies which calls were and were not executed.

Visual Coverage costs $599.
