Desiderata Software Announces PluginMaker

Desiderata Software has announced the release of PluginMaker, a tool for creating plug-ins for Netscape Navigator. SoundCode Announces scCryptoEngine

SoundCode has released scCryptoEngine, a cryptographic engine that provides a framework for crypto-enabling any Windows 95 or NT application. The engine, a set of C++ classes, supports the industry's algorithms for symmetric encryption, public key encryption, hashing, and digital signatures.

With scCryptoEngine any level of encryption is possible, from DES or Triple-DES private keys ranging from 40-bits (exportable) to 168-bits (North America only), and public keys ranging from 512-bits (exportable) to 2,048-bits (North America only). Integrated into Microsoft Foundation Classes, the scCryptoEngine uses Diffie-Hellman/ElGamal public key algorithms and digital signatures, as well as the Digital Signature Standard (DSS).

The scCryptoEngine is available as a static library, DLL, or as a Microsoft Cryptographic Service Provider (CSP). scCryptoEngine costs $230 for a single-developer kit, $600 for a five-developer set, and $760 for a ten-developer set.

Embedded Systems Products Announces RTXC for TMS320C5x and TMS320C2xx

Embedded System Products, Inc. and Spectrum Digital, Inc. have announced RTXC, the real-time operating system (RTOS), supporting the Texas Instruments TMS320C5x and TMS320C2xx fixed-point digital signal processor (DSP) families.

The RTXC design and system resource management structure enables real-time data manipulation and provides a common API for supported processors.

RTXC has over 70 kernel services divided into seven classes. Kernel services include task management, semaphore services, queue services, mailbox and message services, memory management, and timer management. The combination of these kernel services enables such functionality as synchronization of tasks with event occurrence, chronological and prioritized data passing between coordinating tasks, dynamic memory allocation and re-allocation, and the definition and use of programmable timers.

RTXC v3.2, distributed in ANSI C source code and royalty-free, costs $8,500 for supported TI processors. All ESP products come with an initial one-year warranty and a full year of technical support.

Blinkinc's Shrinker Features Direct Execution

Blinkinc has released Shrinker 2.1, a compression and archiving product that improves network performance, minimizes distribution and storage costs, and protects valuable software from decompilers. Shrinker 2.1 compresses 16-bit and real mode DOS programs by 60% or more. These programs execute directly, with no separate decompression step. Any EXE, DLL, or OCX compressed with Shrinker will transparently decompress itself into memory at run time. There is no separate decompression step, so the end-user never has to know it has been shrunk. EXEs compressed with Shrinker often load faster as a because of their reduced size, resulting in no degradation of run-time performance.

A 32-bit version of Shrinker, Shrinker 3.0, was scheduled for release near the time of this issue's printing.
