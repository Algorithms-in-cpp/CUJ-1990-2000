Table 2 Other POSIX file access functions

Function    Description
------------------------------------------------------------
access      Check file existence and permissions
dup         Duplicate a file handle (in-sync with original)
chmod       Change file permissions
fdopen      Associate a FILE * with a file handle
fileno      Extract the file handle from a FILE *
fstat       Get file info (from a FILE *)
stat        Get file info (from a filename)
unlink      Same as remove() on non-UNIX systems
