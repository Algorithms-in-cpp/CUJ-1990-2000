Sanders-Indev Offers XMan Auto Documentation Tool

Sanders-Indev has released XMan, an automatic documentation tool. XMan parses multi-file C projects, capturing the file and line number of every call to any error handler (function or macro), as well as the name and location of the called handler. It also extracts the handler's error message, along with any comments (C or C++) that would explain the error situation. From this data, XMan produces a complete "Errors and warnings" section for a technical manual or a user's guide.

While parsing the .c and .h files, XMan builds a comprehensive chart, graphically showing all caller-called relationships (including macros) in the project. Not only does it show the file and line of every call, it also shows the file and line of each function and macro definition.

XMan runs under DOS 3.1 and later, and costs $195.

PtS Announces OpenNT r2.0

Productivity through Software (PtS) has announced OpenNT r2.0, a native Unix system environment for Windows NT. OpenNT provides integration between Unix and Windows environments on a single machine, giving simultaneous access to pure Unix applications and Windows NT programs.
