Rational Software Corporation has announced an upgrade of their performance profiler Quantify. Software Emancipation Technology Announces ATFx

Software Emancipation Technology, Inc. has announced the addition of Advanced Transformation Framework (ATFx) to its DISCOVER line of products. ATFx employs the technology of tree pattern matching to perform semantic analysis and modifications of source code. It provides the ability to traverse, analyze, and modify source-code syntax trees. The ATFx language allows developers to identify and transform regions of code based on its logical structure.

DISCOVER, with ATFx, can search a system and uncover reusable fragments, dangerous constructs, or areas of suspicious quality. From the DISCOVER browser, developers can launch ATFx search libraries or create ATFx libraries that analyze, evaluate, and propagate throughout the code.
