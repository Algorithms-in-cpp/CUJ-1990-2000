Syware Announces Visual CE

SYWARE, Inc. has announced the availability of Visual CE, a visual development environment for the Windows CE platform. Visual CE is used to create forms-based applications to collect, display, and modify data on Windows CE handheld computers (HPC's). The data can be transferred to and from the desktop via ODBC.

Visual CE provides six common controls: labels, edit boxes, note boxes, check boxes, radio buttons, and drop down lists. Each control is bound to a database field. Controls can be sized, dragged, and positioned as necessary using the desktop WYSIWYG development environment. The built-in calendar and numeric keypad facilitate data entry.

Visual CE Personal Edition costs $299 and includes the Visual CE Development Environment, the Windows CE ODBC driver, and one runtime license. The Developer's Edition costs $1,495 and includes two Visual CE Development Environments, 100 ODBC drivers, and 100 runtime licenses.

DeltaLogic Announces Extended Template Library

DeltaLogic has released the Extended Template Library (XTL) for Win32, an enhanced implementation of the standard C++ library. XTL container classes employ an exception-safe design pattern that eliminates template code bloat, supports multi-threaded programming, and encourages the use of polymorphism. Each container type provides smart iterators that reduce the most common iterator programming errors. XTL allocators have a public interface that lets the containers have per-instance private pools of memory, a feature not available with any other standard library implementation.

XTL supplies extended versions of the standard iostreams and string classes that have many useful operations not provided for by the C++ Standard, such as case-insensitive searching and comparison, and search-based substring replacement.

XTL costs $795 for a single-user license; quantity and educational discounts are available.

TracePoint Technology Releases HiProf

TracePoint Technology has announced HiProf, which supports Microsoft's Visual Studio 97. HiProf's hierarchical profiling enables developers to conduct code analysis on the binaries of 32-bit C and C++ Windows applications. Hierarchical profiling shows each function's self-time, as well as the number of calls taking place between all parent and child functions and the time that activity takes.

HiProf is based on a patented Binary Code Instrumentation (BCI) technology developed by Digital's Western Research Lab. HiProf enables developers to selectively exclude DLLs or portions of their EXE file from profiling in order to drill down on specific portions of an application, or to evaluate third-party components.

Suggested retail for HiProf is $599; a free trial version is also available for download from www.tracepoint.com.
