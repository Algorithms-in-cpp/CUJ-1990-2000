Advanced Visual Systems Unveils OpenViz

Advanced Visual Systems Inc. has launched OpenViz, a suite of components that provides interactive, scalable 2-D and 3-D visual representations of multiple data and chart types. Designed for business applications, OpenViz also supports input of map-related data used in spatial applications.

Silurian Software Debuts Gawp

Silurian Software has introduced Gawp, a software product that provides access to Windows API information and code samples. A feature of Gawp is that C++ code samples are interpreted directly in Internet Explorer or Developer Studio. The Active Samples are integrated into an HTML help system. Using an interpreter means that accurate function timing information is collected and parameters can be modified directly in the browser. The system is an expandable set of small self-contained modules that are distributed directly over the Internet. Current modules available are Introducing GDI (simple graphics calls), Standard GDI (more graphics effects), System Registry access, System information, and MFC GDI (calls to MFC classes).
