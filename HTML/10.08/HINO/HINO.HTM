CUG New Releases

Kenji Hino

Kenji Hino is a member of The C Users' Group technical staff. He holds a B.S.C.S. from McPherson College and an undergraduate degree in metallurgy from a Japanese university. He enjoys playing drums in a reggae band.

The C Users' Group Library is the world's largest holding of public domain and Shareware C software and is composed primarily of submissions by members. C programmers are encouraged to submit software to the Library. Send software on disk (no subdirectories on MS-DOS disks please) to The C Users' Group, 1601 W. 23rd St., Ste. 200, Lawrence, KS 66046-2743. (913) 841-1631.

Updates

CUG326 SoftC Database Library 3 disks)

Soft C, Ltd has provided the v3.0 update of SoftC database library. New features include: dBASEIV support, fully-automatic record and file locking, faster re-index functions, user-extensible Index Expression Evaluator, faster index searches, and more-compact index files. This shareware version includes the full documentation and the small model library for Borland, Zortech, and Microsoft C compilers. To obtain the source code or Windows DLL support, please contact SoftC Limited (612-434-6968).

CUG364 C-ACROSS (2 disks)

Myron Turner (CANADA) has updated his cross-reference programs. The v1.02 update includes minor bug fixes.

CUG343 C Image Processing System (3 disks)

Dwayne Phillips has provided the latest version of the C Image Processing Systems. The disk contains all of the source code for his eight installments of his CUJ article. The disk includes the source code, makefile, header files, and several documentation files.
