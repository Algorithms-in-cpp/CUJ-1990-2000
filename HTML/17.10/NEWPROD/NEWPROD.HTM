New Wave Ships Software Piracy Intervention

New Wave Software has released Software Piracy Intervention (SPI), a product for preventing software piracy and tracking those who attempt it. SPI adds a protective layer to software products that allows only those end users who legitimately purchased the software to install it. No matter how many copies of a protected program get distributed illegally, only the valid purchaser can install the product.

SPI includes passive and active encryption protection. Also, SPI can remotely disable stolen prepackaged software. Additionally, SPI can lock a product to the first computer on which it was successfully installed using the Lock Point feature.

IRealms Launches MagicPatch

IRealms, Inc. has launched MagicPatch, software that enables developers to deploy application updates either automatically or with a click of a button. MagicPatch consists of two main components. First, an analysis program compares existing and new versions of an application and records just the bytes that have changed, then places them on an Internet server where they become available to customers. Second, a program integrated into the application is configured to watch a specific location on the Internet for updates to the application. As soon as the updates are available, MagicPatch will retrieve and apply the changes either automatically or upon demand, as configured by the developer.

MagicPatch is available in two versions: Standard for $599 and Professional for $899. The Professional version includes advanced features such as system file update capabilities, embedded serial number support, and exclusion ranges.
