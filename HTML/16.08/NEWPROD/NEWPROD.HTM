Crossware Releases 8051 C Compiler

Crossware has released a new version of its 8051 C Compiler that takes advantage of the multiple data pointers provided by Siemens, Dallas, and Amtel chips, and the arithmetic unit of the Siemens 80C517/80C537 chip to improve the performance of the generated code.

The standard 8051 microcontroller and the majority of its variants provide only a single 16-bit data pointer with which to access external data and program memory. Access to these memory areas can therefore often involve a large amount of memory swapping as the contents of this data pointer are saved, loaded with a new value, and later restored to the original value. The new Crossware compiler uses the multiple data pointers provided in the support chips to avoid this memory swapping and create faster and smaller code.

C8051NT costs £460 with support for one memory model and £700 with support for two memory models.8051PSDS-A costs £280 with support for one memory model and £520 with support for two memory models.

Enea OSE Systems Unveils OSE Illuminator and OSE MMS

Enea OSE Systems has released OSE Illuminator and OSE Memory Management System (MMS). OSE Illuminator, a suite of tools, debugs and analyzes real-time embedded systems applications during run time. OSE MMS enables software "hot-swapping" to reconfigure and change an embedded system's memory attributes at run time without perturbing the system's operation.

OSE Illuminator integrates three main components that work together to create a comprehensive debugging suite: the System Browser, Evact Handler, and Memory Profiler. The System Browser gives a clear overview of an entire distributed system, with each CPU target represented. The Evact (for "event" and "action") Handler is used to debug interprocess communications and other events in an OSE application. The Memory Profiler allows users to analyze and debug run-time memory usage in an OSE system even if the system uses many distributed processors.

OSE MMS integrates the OSE kernel memory management features with both a hardware memory management unit (MMU) and the OSE Program Handler, enabling run-time loaded programs to be memory protected. OSE MMS is available for systems based on the OSE fault-tolerant, real-time operating System (RTOS).

The OSE Illuminator costs $2,000; the OSE MMS costs $2,500.

For more information, contact Enea OSE Systems, Inc., +1.214.346.9339; fax +1.214.346.9344; info@enea.com; www.enea.com.
