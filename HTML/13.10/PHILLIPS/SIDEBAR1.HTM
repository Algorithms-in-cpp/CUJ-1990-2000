Spatial Bi-linear Interpolation

Q(a) = P1 + (P2 - P1)a(1)

R(a)=P4 + (P3 - P4)a(2)

P(a,b) = Q + (R - Q)b(3)

P(a,b) = P1 + (P2 - P1)a + [(P4 + (P3 - P4)a) - (P1 + (P2 - P1)a)]b(4)

P(a,b)=P1 + (P2 - P1)a + (P4 - P1)b + (P1 - P2 + P3 - P4)ab(5)

P(x) = x1 + (x2 - x1)a + (x4 - x1)b + (x1 - x2 + x3 - x4)ab(6)

P(y) = y1 + (y2 - y1)a + (y4 - y1)b + (y1 - y2 + y3 - y4)ab(7)
