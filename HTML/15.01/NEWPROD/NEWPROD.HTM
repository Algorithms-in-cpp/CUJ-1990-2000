ICS Ships BX PRO Cross-Platform Environment

Integrated Computer Solutions has announced Builder Xcessory PRO (BX PRO), an integrated product suite for visual, object-oriented development. BX PRO supports both Java and C/C++ development. Using BX PRO, enterprise developers can develop applications on UNIX workstations and deploy applications across multiple platforms. BX PRO enables cross-platform applications to be built in Java, or, with an optional software development kit, allows applications built in C/C++ to be deployed to Windows-based systems.

BX PRO, a WYSIWYG visual development environment, consists of Builder Xcessory; ViewKit ObjectPak, a C++ reuse framework that can be tailored to accommodate the specific needs of individual businesses; and EnhancementPak, a library of 27 reusable components, including business graphs and user interface controls. Platforms supported by BX PRO include UNIX systems from Sun, HP, IBM, DEC, and SGI. Java applications built with BX PRO can run on systems that support the Java platform, including Windows 95, NT, and Apple MacOS. An optional software development kit allows applications built with C/C++ to be deployed on Windows 95 and Windows NT.

A floating license with one year of support for BX PRO costs $6,495.

Dyad Releases M++ v6.0

Dyad Software has announced the release of the C++ math library and array language software package, M++ v6.0. This C++ math library works in conjunction with many popular C++ compilers on PC and UNIX platforms. A number of advanced modules extend the functionality of M++ into specialized operations such as optimization, statistical operations, and finite element modeling.

Version 6.0 features a comprehensive exception-handling package with both error and warning messages. Version 6.0 also introduces a new set of "safe" functions that test data arrays prior to operations, such as log or square root, to prevent computational failures due to out-of-range values. The library is designed to allow Windows programmers to easily create their own popup windows based on the exceptions. Additional features added to M++ include root finders and interpolation/extrapolation functions.

M++ v6.0 is available for $495 for PC operating systems and $795 for single-user systems. Source code is available. System availability and multi-user pricing is available upon request. Demo software is available on the Dyad Software website.
