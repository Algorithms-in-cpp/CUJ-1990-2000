Accelerated Technology Announces Nucleus C++ RTOS

Accelerated Technology has announced Nucleus C++, a real-time operating system (RTOS) that provides direct support for multitasking and object-oriented methodologies. Nucleus C++ provides active objects and other object-oriented mechanisms to synchronize and otherwise manage the existence and behavior of active objects within a real-time software system. Direct support for multitasking is achieved by encapsulating execution into objects. Standard C++ mechanisms are used for creating, using, and ultimately destroying objects. The executable portions of the active objects are created and managed automatically.

Nucleus C++ addresses common performance and management concerns about migrating to an object-oriented design methodology. The underlying RTOS is the Nucleus PLUS real-time operating system. Nucleus PLUS is a small, fast, and portable C multitasking kernel. Common concerns regarding code size and execution speed are addressed by tight coupling to the RTOS. Nucleus C++ requires Nucleus PLUS.

Nucleus C++ costs $2,500. Nucleus PLUS costs from $5,000 to $9,000 depending on the platform.

Azalea Announces carrick v1.1

Azalea Software, Inc. has begun shipping carrick v1.1, an encryption utility that provides private key encryption. Unlike most other computer security products, carrick is based on the new Blowfish encryption algorithm. While others debate the merits of 40- or 56-bit key lengths, carrick allows keys up to 448 bits long. Because of the strength of the underlying algorithm, the U.S. government won't allow carrick to be sold outside the U.S.

The Windows version runs under Windows 3.x, Windows 95, and Windows NT. Improvements to the Windows product include drag-and-drop support from the File Manager, the option to save the two halves of a key to disk, notification of potentially weak keys, and a helpful rotating tip of the day highlighting sound security practices.

carrick costs $159, or $199 for two copies.

Continuus Ships Continuus/CM v4.2

Continuus Software Corporation has begun shipping Continuus Change Management Suite v4.2. Continuus/CM v4.2 tracks changes through the entire set of files involved in a given programming task.

Continuus' task-based CM capability assumes the burden of tracking and operating on related sets of changes, which means the following: it eliminates the need for developers to maintain lists of files they've edited to complete each enhancement and bug fix; it frees the developer from checking in files one at a time and eliminates the failed compilations common in file-based CM systems when programmers forget to check in all of the files edited to complete a task; it allows changes related to an enhancement or bug fix to be backed out of a software release with a mouse click, useful when incomplete tasks are holding up software delivery; and it enhances software quality through detection of incomplete or inconsistent sets of changed files.

NewCode Ports NCi C++ Interpreter to Windows NT

NewCode Technology has ported NCi to Windows NT. NCi, a standards-compliant C++ interpreter for embedding in end-user applications, provides end-users of Windows NT applications the ability to extend functionality in their C++ products without opening up large portions of the development environment.
